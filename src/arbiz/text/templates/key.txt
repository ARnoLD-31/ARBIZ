Инструкция по активации для Windows 10:
 1. Нажмите сочетание клавиш Win + I.
 2. Перейдите в раздел "Обновление и безопасность" → "Активация".
 3. Нажмите на "Изменить ключ продукта".
 4. Введите ключ активации и нажмите "Далее".

Инструкция по активации для Windows 11:
 1. Нажмите сочетание клавиш Win + I.
 2. Перейдите в раздел "Система" → "Активация".
 3. Нажмите на "Изменить ключ продукта" → "Изменить".
 4. Введите ключ активации и нажмите "Далее".
